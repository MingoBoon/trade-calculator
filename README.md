# Position Size, Risk & True-Cost Calculator

A single-file web tool for sizing share/ETF trades to a fixed % risk and seeing the real cost after broker fees, FX, stamp duty and spread.

**Live version:** _(add your GitHub Pages link here once published)_

## Files
- `index.html` — the calculator (this is what GitHub Pages serves)
- `Position-Size-Risk-Calculator.html` — working copy (same content)
- `UK-Brokerage-Cost-Deep-Dive.md` — broker fee reference behind the cost engine

## How to update it
Edit `index.html` on GitHub (pencil icon) → Commit changes. The live site refreshes in ~1 minute.

## Notes
- Runs entirely in the visitor's browser. No data is stored or sent anywhere.
- Live prices come from a public delayed feed via a CORS proxy and may occasionally be blocked — the manual price entry always works.
- Educational tool, not financial advice.
